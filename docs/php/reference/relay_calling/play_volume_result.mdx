# Relay.Calling.PlayVolumeResult

This object is returned by [`volume`](php/reference/relay_calling/play_action.mdx#volume) method and represents the final result of a volume control operation.


## Properties

|    |   |   |
| -: | - | - |
| `successful` | <span class="method-prop-type">boolean</span> | Whether the playing volume has been changed successfully. |

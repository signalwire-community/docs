# Relay.Event

This object represents the last Relay event that completed an operation on the Call.

## Properties

|           |                                              |                                     |
| --------: | -------------------------------------------- | ----------------------------------- |
|    `name` | <span class="method-prop-type">string</span> | The event name.                     |
| `payload` | <span class="method-prop-type">object</span> | Raw JSON object of the Relay event. |

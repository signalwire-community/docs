# Relay.Calling.PlayResult

This object returned from one of _synchronous_ play methods that represents the final result of a playing action.

## Methods

<div class="method-definitions"></div>

### getEvent

Returns the last Relay Event arrived for this operation.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

[`Relay.Event`](relay/server-sdk/php/reference/relay_event.mdx) - Last Relay Event.

**Examples**

```php title="Start playing an audio file and then grab the last event occurred."
<?php

$call->playAudio('https://cdn.signalwire.com/default-music/welcome.mp3')->done(function($result) {
  $event = $result->getEvent();
  // Inspect $event->payload ..
});
```

### isSuccessful

Return `true` if the playing has succeeded, `false` otherwise.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`boolean` - Whether the playing has finished successfully.

**Examples**

```php title="Start the recording and then check if it has ended successfully."
<?php

$call->playAudio('https://cdn.signalwire.com/default-music/welcome.mp3')->done(function($result) {
  if ($result->isSuccessful()) {
    // Do other things...
  }
});
```

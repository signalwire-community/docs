# Relay.Calling.DetectAction

This object returned from one of _asynchronous_ detect methods that represents a running detector on the call.

## Methods

<div class="method-definitions"></div>

### getControlId

Return the UUID to identify the action.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.1+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`string` - UUID to identify the action.

**Examples**

```php title="Start a detector and print the controlId."
<?php

$call->detectMachineAsync()->done(function ($action) {
  echo $action->getControlId();
});
```

### getPayload

Return the payload sent to Relay to initiate the request. Useful to inspect what you sent to perform this action.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.1+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`Object` - Payload sent to Relay.

**Examples**

```php title="Start a detector and print out the payload."
<?php

$call->detectDigitAsync()->done(function($detectResult) {
  print_r($action->getPayload());
});
```

### getResult

Returns the final detector result.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.1+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

[`Relay.Calling.DetectResult`](relay/server-sdk/php/reference/relay_calling/detect_result.mdx) - Final detector result.

**Examples**

```php title="Trying detecting DTMF and grab the result when it's completed."
<?php

$call->detectDigitAsync()->done(function($action) {
  // .. later in the code since it's an async method
  if ($action->isCompleted()) {
    $detectResult = $action->getResult();
  }
});
```

```php title="Trying detecting DTMF and check if it has finished."
  }

```

### stop

Stop the action immediately.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.1+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`React\Promise\Promise` - Promise object that will be fulfilled with a [`Relay.Calling.StopResult`](relay/server-sdk/php/reference/relay_calling/stop_result.mdx) object.

**Examples**

```php title="Trying detecting a machine and then stop the action."
<?php

$call->detectMachineAsync()->done(function ($action) {
  // For demonstration purposes only..
  $action->stop()->done(function($stopResult) {

  });
});
```

# Relay.Calling.PlayPauseResult

This object is returned by [`pause`](relay/server-sdk/php/reference/relay_calling/play_action.mdx#pause) method and represents the final result of a play pause operation.

## Properties

|              |                                               |                                                   |
| -----------: | --------------------------------------------- | ------------------------------------------------- |
| `successful` | <span class="method-prop-type">boolean</span> | Whether the playing has been paused successfully. |

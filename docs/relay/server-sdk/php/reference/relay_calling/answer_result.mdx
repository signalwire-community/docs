# Relay.Calling.AnswerResult

This object returned from the [`answer`](relay/server-sdk/php/reference/relay_calling/call.mdx#answer) method.

## Methods

<div class="method-definitions"></div>

### getEvent

Returns the last Relay Event arrived for this operation.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

[`Relay.Event`](relay/server-sdk/php/reference/relay_event.mdx) - Last Relay Event.

**Examples**

```php title="Start recording in stereo mode and grab the result when it's completed."
<?php

$call->answer()->done(function($result) {
  $event = $result->getEvent();
  // Inspect $event->payload ..
});
```

### isSuccessful

Return `true` if the call was answered without errors, `false` otherwise.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`boolean` - Whether the call has been answered successfully.

**Examples**

```php title="Start the recording and then check if it has ended successfully."
<?php

$call->answer()->done(function($result) {
  if ($result->isSuccessful()) {
    // $call is now active...
  }
});
```

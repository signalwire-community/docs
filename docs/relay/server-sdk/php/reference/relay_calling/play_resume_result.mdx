# Relay.Calling.PlayResumeResult

This object is returned by [`resume`](relay/server-sdk/php/reference/relay_calling/play_action.mdx#resume) method and represents the final result of a play resume operation.

## Properties

|              |                                               |                                               |
| -----------: | --------------------------------------------- | --------------------------------------------- |
| `successful` | <span class="method-prop-type">boolean</span> | Whether the playing has resumed successfully. |

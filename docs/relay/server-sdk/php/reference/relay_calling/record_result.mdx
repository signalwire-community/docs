# Relay.Calling.RecordResult

This object returned from [`record`](relay/server-sdk/php/reference/relay_calling/call.mdx#record) method that represents the final result of a recording.

## Methods

<div class="method-definitions"></div>

### getDuration

Returns the duration of the recording in seconds.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`number` - Duration of the recording in seconds.

**Examples**

```php title="Start recording and use the duration."
<?php

$params = [
  'stereo' => true
];
$call->record($params)->done(function($result) {
  if ($result->isSuccessful()) {
    $duration = $result->getDuration();
  }
});
```

### isSuccessful

Return `true` if the recording succeeded, `false` otherwise.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v2.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`boolean` - True/False accordingly to the state.

**Examples**

```php title="Start the recording and then check if it has ended successfully."
<?php

$params = [
  'stereo' => true
];
$call->record($params)->done(function($result) {
  if ($result->isSuccessful()) {
    // Recording completed with success
  }
});
```

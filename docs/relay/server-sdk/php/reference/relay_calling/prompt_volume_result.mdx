# Relay.Calling.PromptVolumeResult

This object is returned by [`volume`](relay/server-sdk/php/reference/relay_calling/prompt_action.mdx#volume) method and represents the final result of a volume control operation.

## Properties

|              |                                               |                                                   |
| -----------: | --------------------------------------------- | ------------------------------------------------- |
| `successful` | <span class="method-prop-type">boolean</span> | Whether the volume has been changed successfully. |

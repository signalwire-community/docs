# React Library for SignalWire

SignalWire Community's React Library enables the seamless integration of
SignalWire APIs with React applications. With the use of high quality components
and hooks, you can get started in minutes.

 - **npm:** [@signalwire-community/react](https://www.npmjs.com/package/@signalwire-community/react) ![@signalwire-community/react](https://img.shields.io/npm/v/@signalwire-community/react)
 - **GitHub:** [signalwire-community/react](https://github.com/signalwire-community/react)

## Installation

Install the package using npm or yarn:

```shell npm2yarn
npm install @signalwire-community/react
```

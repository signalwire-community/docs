# LocalStream

This React component gives you an easy way to display the local stream
transmitted in a room session.

## Import

```jsx
import { LocalStream } from "@signalwire-community/react";
```

## Props

<div className="lightTable" />

| Name          | Type                  | Default | Description       |
| :------------ | :-------------------- | :------ | :---------------- |
| `roomSession` | `RoomSession`         |         | The RoomSession.  |
| `style`       | `React.CSSProperties` |         | Style properties. |

## Example

```jsx
<LocalStream roomSession={roomSession} style={{ height: 150 }} />
```

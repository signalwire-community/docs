# RoomPreview

This React component gives you an easy way to display the preview of a video room.

## Import

```jsx
import { RoomPreview } from "@signalwire-community/react";
```

## Props

<div className="lightTable" />

| Name         | Type                  | Default | Description                                                       |
| :----------- | :-------------------- | :------ | :---------------------------------------------------------------- |
| `previewUrl` | `string`              |         | The URL to the room preview.                                      |
| `loadingUrl` | `string`              |         | The URL to an image to display while the room preview is loading. |
| `style`      | `React.CSSProperties` |         | Style properties.                                                 |

## Example

```jsx
<RoomPreview
  previewUrl={previewUrl}
  loadingUrl="https://swrooms.com/swloading.gif"
  style={{ height: 150 }}
/>
```

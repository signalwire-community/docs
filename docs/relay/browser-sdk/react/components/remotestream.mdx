# RemoteStream

This React component gives you an easy way to display the remote stream coming from a room session.

Please note that this will not include local overlays, so you may notice a
slight latency in your own camera stream.

## Import

```jsx
import { RemoteStream } from "@signalwire-community/react";
```

## Props

<div className="lightTable" />

| Name          | Type                  | Default | Description       |
| :------------ | :-------------------- | :------ | :---------------- |
| `roomSession` | `RoomSession`         |         | The RoomSession.  |
| `style`       | `React.CSSProperties` |         | Style properties. |

## Example

```jsx
<RemoteStream roomSession={roomSession} style={{ height: 150 }} />
```

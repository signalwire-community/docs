# LamlRepository

The `LamlRepository` contains method used to create,get, update and delete XML bins using the Java Wrapper Library.

## Constructors

### constructor

- **new LamlRepository**(String projectId, String apiToken, String spaceName)

**Parameters**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `projectId` | `String` | ProjectId gotten from your Signalwire space e.g `a10d8a9f-2166-4e82-56ff-118bc3a4840f` |
| `apiToken` | `String` | Api token generated from your Signalwire dashboard e.g `PT9e5660c101cd140a1c93a0197640a369cf5f16975a0079c9`|
| `spaceName` | `String` | Name of your space e.g `testing` |

### create

▸**calls**(String name, String content) : [Bin](java/reference/xml/classes/Bin.mdx)

This method is used to create an XML Bin that is to be used on your SignalWire space.

**Parameters**

| Name | Type | Description |
|---|---|---|
| `name` | String | Friendly name used to recognise your XML Bin |
| `content` | String | specified xml tags or content used to represent a Call, Message or Fax. Check out [VoiceResponse](java/reference/xml/classes/VoiceResponse.mdx), [MessageResponse](java/reference/xml/classes/MessageResponse.mdx), [FaxResponse](java/reference/xml/classes/FaxResponse.mdx) on how to generate XML bins |


**Usage**

Firstly, Generate an XML using [VoiceResponse](java/reference/xml/classes/VoiceResponse.mdx)

```java

// Attribute to get Say up and running
HashMap<String, String> attribute = new HashMap<>();
attribute.put("voice", "woman");
attribute.put("language", "fr-CA");
attribute.put("loop", "5");

// Using the VoiceResponse to generate the xml
VoiceResponse generate = new VoiceResponse.Builder()
            .say(attribute, "Hello world")
            .build();

// Print the response
System.out.println(generate.getResponse());

// Create the new Bin
Bin response = repository.create("test-bin", generate.getResponse());

// Print out the SID
System.out.println(response.sid);

```

#### Returns

[Bin](java/reference/xml/classes/Bin.mdx)

# VoiceRepository

The `VoiceRepository` contains method used to create,get, update and delete Voice calls using the Java Wrapper Library.

## Constructors

### constructor

- **new VoiceRepository(String projectId, String apiToken, String spaceName)**

**Parameters**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `projectId` | `String` | ProjectId gotten from your Signalwire space e.g `a10d8a9f-2166-4e82-56ff-118bc3a4840f` |
| `apiToken` | `String` | Api token generated from your Signalwire dashboard e.g `PT9e5660c101cd140a1c93a0197640a369cf5f16975a0079c9`|
| `spaceName` | `String` | Name of your space e.g `testing` |

### create
▸**create**([VoiceBuilder](java/reference/voice/classes/VoiceBuilder.mdx) builder): [Call](java/reference/voice/classes/Call.mdx)

This method enables user initiate a call using the Signalwire API, the `create` method takes a `VoiceBuilder` object and creates a phone call using the URL of your xml bins or application id.

|Name|Type|Description|
|---|----|----|
|`builder`| [VoiceBuilder](java/reference/voice/classes/VoiceBuilder.mdx) | a `builder` class used to populate information to make a call. |

**Example**

```java
VoiceBuilder builder = new VoiceBuilder).Builder()
                        .to("to-number")
                        .from("from-number")
                        .url("xml-bin-url") //Url generated after creating xml bin
                        .build();

// Make the voice call
Call response = voiceRepository.create(builder);

// print the call SID
System.out.println(response.sid);
```

#### Returns
[Call](java/reference/voice/classes/Call.mdx)
